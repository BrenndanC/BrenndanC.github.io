Attendance Collection

This project was the first application I developed. It is given a file of attendance logs, and will return the names of students along with any relevant information asked for. Features include but are not limited to:

  • List of late students
  • Attendance for a specific student
  • List of students absent
  • List of all student entry times

    • For the application to work, the text file provided has to be within a certain format. Should the file deviate from this format, the application will fail to work. Here is a video that demonstrates the capabilities of the application as well as how to use it.
